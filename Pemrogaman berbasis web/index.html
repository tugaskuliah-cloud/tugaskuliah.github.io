MSIB KAMPUS MERDEKA

Program MSIB Kampus Merdeka memberikan saya semangat baru untuk belajar dan mencari pengalaman di luar kampus. Proses Seleksi untuk mengikuti MSIB terbilang cukup panjang, dimulai dari seleksi berkas yang cukup banyak, kemudian dilanjutkan dengan tes dari masing-masing mitra. Banyak perusahaan besar tergabung dalam mitra MSIB, sehingga mendorong saya untuk mendaftar progra ini.

RevoU adalah salah satu mitra yang menarik perhatian saya karena programnya relevan dengan bidang pekerjaan yang banyak dicari. Testimoni positif dan kolaborasi dengan Amazon Web Services (AWS) menambah daya tariknya. Program di RevoU memberikan pengetahuan dan keterampilan baru di bidang Data Analyst, dengan materi up-to-date dan pengajar berpengalaman.

Selama mengikuti program di RevoU, saya mendapatkan banyak pengetahuan dan keterampilan baru yang sangat bermanfaat, terlebih di bidang Data Analyst. Materi yang disampaikan sangat up-to-date, dan lecture yang mengajar juga memiliki pengalaman yang luas di industri dan perusahaan besar. Selain itu, kesempatan untuk berkolaborasi dengan AWS memberikan saya wawasan mendalam tentang teknologi cloud computing.

Setiap modul disertai dengan assignment dan ada juga Capstone Project sebagai proyek akhir yang menguji pemahaman dan meningkatkan kemampuan analisis serta problem-solving. Melalui Capstone Project dapat memberika pengalaman ini mengasah hard skill dalam kerja sama tim.

Saya merasa sangat beruntung bisa menjadi bagian dari program ini. Setiap hari saya belajar hal baru dan semakin percaya diri untuk menghadapi tantangan. Teruslah berusaha dan jangan ragu!. Pengalaman berharga dan kesempatan besar menanti kalian.
